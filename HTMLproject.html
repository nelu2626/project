Forms in HTML

HTML offers various form controls that allow users to interact with web pages some of the common form controls include:
  • Input:Text:<input type="text">for single line text input
  • Password:<input type="password">for password input(text is hidden)
  • Checkbox:<input type="checkbox">for checkboxes
  • Radio:<input type="radio">for radio button(choose any one)
  • Number:<input type="number">for numeric input
  • E-mail:<input type="e-mail">for e-mail input(verifies email format)
  • Date:<input type="date">for choosing a date
  • File:<input type="file">for file uploads
  • Hidden:<input type="hidden">for hidden input feilds
  • Color:<input type="color">for selecting a color
  • Range:<input type="range">for selecting a value from a range
  • Search:<input type="search">for search input
  • Tel:<input type="tell">for telephone number input
  • URL:<input type="URL">for url input

  • <select>:creates a dropdown list
  • <option>:Used within <select> to define options in a dropdown.
  • Textarea:<textarea>:Creates a multi-line text input area.
  • Button:<input type="button">:creates a clickable button.
  • <input type="submit">:Submit a form
  • <Label>: Associates a label with a form control
  • <feildset>Groups related elements in a form
  • <legend>:Provides a caption for a <feildset> element
    • These form controls help in collecting and processing user input on web pages.Each control has its own specific purpose and behaviour providing a wide range of options for creating interactive forms
